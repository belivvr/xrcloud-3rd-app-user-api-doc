Avatars

API Address​

[GET] https://3rd-api.xrcloud.belivvr.com/v1/apps/{appId}/resources/avatars

Description​

Get all avatar registered in your Application.

Headers​

Authorization: Bearer refresh_token

Response​

Http Status: 200 OK
Content-Type: application/json
